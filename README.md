# SehaPracto - Responsive and Multi-language Web Application


## Overview

SehaPracto is a web application that started as a PDF design and was transformed into a responsive and multi-language site in just 24 hours. This project was developed by Alaa Abo Arshid using various modern web development tools and technologies.

## Features

- **Responsive Design**: SehaPracto is designed to adapt to different screen sizes, ensuring a seamless user experience on various devices, including desktops, tablets, and smartphones.

- **Multi-language Support**: The web application is equipped with multiple language options, making it accessible and user-friendly for a diverse audience.

## Tools and Technologies Used

The following tools and technologies were utilized in the development of SehaPracto:

- [Vue.js](https://vuejs.org/): A progressive JavaScript framework for building user interfaces.

- [Nuxt.js](https://nuxtjs.org/): A higher-level framework built on top of Vue.js that simplifies the development of server-side rendered (SSR) applications.

- [Vuetify](https://vuetifyjs.com/): A Material Design component framework for Vue.js that enables developers to create visually appealing and consistent user interfaces.

- [@nuxtjs/i18n](https://i18n.nuxtjs.org/): A Nuxt.js module for easy internationalization (i18n) of applications, enabling seamless translations and localization.

## Deployment

SehaPracto is deployed on [Vercel](https://vercel.com/) using a Node.js server, ensuring a fast and reliable hosting environment.

## Live Demo

You can visit the live demo of SehaPracto by clicking [here](https://seha-practo.vercel.app/). Explore the responsive design and try out the multi-language support to experience the full capabilities of this web application.
